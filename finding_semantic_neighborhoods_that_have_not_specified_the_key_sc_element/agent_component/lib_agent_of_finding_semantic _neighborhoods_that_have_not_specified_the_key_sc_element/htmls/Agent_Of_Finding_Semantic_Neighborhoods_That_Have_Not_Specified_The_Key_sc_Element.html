Задачей агента поиска подклассов семантических окрестностей, у которых не указан ключевой sc-элемент является нахождение семантических окрестностей, у которых не указан ключевой sc-элемент в соответствующую структурах. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска семантических окрестностей, у которых не указан ключевой sc-элемент . Единственным аргументом запроса является структура в которой необходимо найти семантические окрестности, у которых не указан ключевой sc-элемент.
