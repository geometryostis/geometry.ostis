Задачей агента назначения менеджера* является назначение пользователя менеджером* БЗ . Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу назначения менеджером* БЗ. Единственным аргументом запроса является пользователь, которого необходимо назначить менеджером* БЗ.
