Операторы класса scp-оператор синхронизации выполнения scp-программы предназначены для синхронизации переходов между scp-операторами в рамках одной scp-программы и не описывают никаких непосредственных преобразований sc-памяти.

Операторы данного класса могут использоваться, например, когда возникает необходимость дождаться завершения выполнения нескольких scp-операторов, выполнявшихся параллельно в рамках одной из веток scp-программы, порожденной каким-либо scp-оператором проверки условий.
