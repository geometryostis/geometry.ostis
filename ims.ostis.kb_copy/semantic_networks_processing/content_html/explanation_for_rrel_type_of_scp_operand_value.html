Данная таблица поясняет возможные комбинации ролевых отношений тип scp-операнда' и тип значения scp-операнда':

scp-операнд с заданным значением' scp-операнд со свободным значением'
scp-константа' Разрешено, может быть опущено Запрещено
scp-переменная' Разрешено, значение останется неизменным Разрешено, значение переменной будет изменено либо потеряно
