SCs-код. Уровень 1 аналогичен SCg-коду уровня 1 и представляет собой простейшее универсальное средство кодирования (представления) любого sc-графа в линейном виде, т.е. в виде цепочки (строки) символов. При этом каждому sc-элементу представляемого sc-графа взаимно-однозначно ставится в соответствие его строковый идентификатор (имя), что делает необходимым специально придумывать имена для всех (!) используемых sc-элементов, входящих в состав представляемого sc-графа.
