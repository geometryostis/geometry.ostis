Во множество нерешенных sc-задач входят формулировки задач, решение которых на данный момент не может быть получено каким-либо sc-агентом или коллективом sc-агентов в рамках системы, то есть, не может быть достигнута целевая ситуация, описанная в формулировке данной задачи.

Можно выделить две основные причины, по которым может сложиться указанная ситуация:

  • sc-задача сформулирована некорректно;
  • формулировка sc-задачи корректна и понятна системе, однако решение данной задачи в текущий момент не может быть получено за удовлетворительные с точки зрения системы или пользователя сроки.

Для того, чтобы оценить трудоемкость решения задачи и объективную возможность получить решение в текущем состоянии системы необходимо наличие в системе специально ориентированных на это sc-агентов.

Для конкретизации факта некорректности формулировки задачи можно выделить ряд более частных классов нерешенных sc-задач, например:

  • sc-задача, формулировка которой противоречит другим знаниям системы (например, не выполняется неравенство треугольника);
  • sc-задача, в формулировке которой использованы понятия, неизвестные системе;
  • sc-задача, решение которой невозможно из-за недостаточности данных (например, найти площадь треугольника по двум сторонам);
  • и другие.
