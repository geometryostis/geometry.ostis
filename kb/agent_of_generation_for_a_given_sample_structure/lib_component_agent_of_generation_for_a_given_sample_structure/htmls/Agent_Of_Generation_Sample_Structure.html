Задачей агента генерации структуры по заданному образцу является генерация заданной структуры. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу генерации структуры по заданному образцу. Первым аргументом запроса является сущность, для которой необходимо применить обобщённую спецификацию или необходимую обобщённую спецификацию. Вторым аргументом запроса является обобщённая спецификация или необходимая обобщённая спецификация, которую необходимо применить. Результатом выполнения команды является сгенерированная структура, соответствующий поданной на вход спецификации.
