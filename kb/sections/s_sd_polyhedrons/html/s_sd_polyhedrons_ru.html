Многогранником называется геометрическое тело, ограниченное конечным числом плоских многоугольников, из которых любые два смежных не лежат в одной плоскости.

Ребром многогранника называется каждая из сторон многоугольника, ограничивающего многогранник.

Вершиной многогранника называется каждая из вершин любого многоугольника, ограничивающего многогранник.

Выпуклым называется многогранник, который лежит по одну сторону от плоскости каждой своей грани. Если это условие не выполняется, то многогранник называется невыпуклым.

Формула Эйлера для выпуклого многогранника: N - L + F = 2, где N - число sc_element sys_idtf="nrel_vertex">вершин данного многогранника, L - число его ребёр, F - число граней.

Диагональю многогранника называется отрезок, соединяющий две вершины многогранника, не принадлежащих одной грани.

Полной поверхностью многогранника называется фигура, образованная всеми его гранями.

Площадью полной поверхности многогранника называется сумма площадей всех его граней.

Секущей плоскостью многогранника, называется такая плоскость, по обе стороны от которой есть точки данного многогранника.

Сечением многогранника называется фигура, состоящая из всех точек, которые являются общими для данного многогранника и секущей плоскости. Сечение многогранника есть многоугольник, лежащий в секущей плоскости, сторонами которого являются отрезки, по которым секущая плоскость, пересекает грани многогранника. для построения изображения сечения многогранника следует построить точки пересечения секущей плоскости с ребрами этого многогранника и соединитькаждые двее из них, лежащие в одной грани.
