Задачей агента удаления множества структур для визуализации является удаления множества структур для визуализации. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей удаления множества структур для визуализации. Единственным аргументом запроса является структура, которая принадлежит множеству структур для визуализации.
