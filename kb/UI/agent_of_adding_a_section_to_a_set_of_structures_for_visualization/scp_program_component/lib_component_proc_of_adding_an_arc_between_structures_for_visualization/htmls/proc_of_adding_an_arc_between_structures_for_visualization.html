Задачей scp-программы добавления дуги между двумя структурами для визуализации является добавления дуги между двумя структурами для визуализации. Первым и вторым входным параметрами данной scp-программы являются структуры, между которыми генерируется дуга с отношением порядок визуализации* и порядок следования*.
