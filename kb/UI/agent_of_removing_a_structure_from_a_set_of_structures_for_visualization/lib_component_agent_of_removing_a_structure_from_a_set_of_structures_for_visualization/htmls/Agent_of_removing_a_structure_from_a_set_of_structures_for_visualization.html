Задачей агента удаления структуры из множества структур для визуализации является удаления структуры из множества структур для визуализации. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей удаления структуры из множества структур для визуализации. Единственным аргументом запроса является структура, которую необходимо удалить множества структур для визуализации.
