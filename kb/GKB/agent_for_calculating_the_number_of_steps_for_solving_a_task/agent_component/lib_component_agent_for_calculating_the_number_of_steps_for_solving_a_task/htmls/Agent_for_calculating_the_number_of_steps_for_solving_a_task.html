Задачей агента подсчёта количества шагов решения задачи является поиск в базе знаний действий, необходимых для решения задачи, и подсчета их количества. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей команда для подсчёта количества шагов решения задачи. Аргументом запроса является либо утверждение, либо определение, либо действие.
