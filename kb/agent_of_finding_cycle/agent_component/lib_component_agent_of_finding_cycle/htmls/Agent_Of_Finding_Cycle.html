Задачей агента поиска циклов является поиск цикла для заданного отношения порядка. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска цикла для любого отношения порядка . В данном агенте предусмотренна возможность ограничения области определения до конкретной предметной области. Для рядовых разработчиков удобней использовать этот агент в рамках какой-то предметной области, то есть использовав два входных агрумента (первым аргументом является любое отношение порядка, вторым - предметная область, в рамках которой происходит поиск цикла). Для администраторов уместней использовать этот агент в рамках всей базы знаний.
