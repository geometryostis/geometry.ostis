Задачей scp-программы поиска цикла является поиск цикла в рамках области действия scp-программы. В данной scp-прогрымме предусмотренна возможность ограничения области определения до конкретной предметной области. Для рядовых разработчиков удобней использовать эту scp-программу в рамках какой-то предметной области, то есть использовав два входных агрумента (первым аргументом является любое отношение порядка, вторым - предметная область, в рамках которой происходит поиск цикла). Для администраторов уместней использовать эту scp-программу в рамках всей базы знаний. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: атрибутами 1'.
