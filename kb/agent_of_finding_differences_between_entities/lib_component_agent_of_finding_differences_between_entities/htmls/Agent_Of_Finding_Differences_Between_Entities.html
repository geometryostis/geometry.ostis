Задачей агента поиска различий между двумя сущностями является поиск всех структур, в которых учавствует первый(второй) агрумент и не учавствует второй(первый). Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска различий между заданным сущностями. Первым и вторым аргументами запроса являются сущности, различия между которыми необходимо выявить. Результатом выполнения команды являются все найденные различия между двумя элементами.
