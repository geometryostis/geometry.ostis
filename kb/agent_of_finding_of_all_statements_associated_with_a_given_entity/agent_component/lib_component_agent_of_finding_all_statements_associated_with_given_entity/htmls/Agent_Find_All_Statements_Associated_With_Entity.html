Задачей агента вывода всех утверждений, ассоциированных с заданной сущностью является поиск и вывод всех утверждений и определений, ассоциированных с заданной сущностью. Т.е. вывод всех утверждений и определений, непосредственно связанных с заданной сущностью и определяющих сущности, с которыми заданная сущность связанна иерархией. По иерархии агент перемещается с помощью отношения включение и разбиение/ Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вывода всех утверждений, ассоциированных с заданной сущностью.
