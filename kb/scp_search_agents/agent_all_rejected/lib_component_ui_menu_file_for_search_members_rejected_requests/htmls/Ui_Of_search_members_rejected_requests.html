Команда поиска отклоненных пользователем запросов предназначена для поиска отклоненных пользователем запросов. Единственным аргументом команды является автор, для которого необходимо найти все отклоненные им предложения. Результатом выполнения команды является структура, содержащая все отклоненные предложения.
