Команда поиска утвержденных пользователем запросов предназначена для поиска утвержденных пользователем запросов. Единственным аргументом команды является автор., для которого необходимо найти все утвержденные им предложения. Результатом выполнения команды является структура, содержащая все утвержденные предложения.
