Команда поиска не рассмотренных пользователем запросов предназначена для поиска не расмотренных пользователем запросов. Единственным аргументом команды является автор, для которого необходимо найти все не рассмотренные им предложения. Результатом выполнения команды является структура, содержащая все не рассмотренные предложения.
