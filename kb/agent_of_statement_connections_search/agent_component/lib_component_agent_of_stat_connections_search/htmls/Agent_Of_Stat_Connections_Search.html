Задачей агента поиска связи исходного утверждения с другими утверждениями является нахождение утверждений для заданного утверждения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска связей утверждения с другими утвеждениями. Аргументами запроса являются утверждение, для которого осуществляется поиск и узел, в которому прикрепляется ответ.
