Задачей scp-программы добавления элемента предметной области в теоретико-множественную онтологию является добавление элемента предметной области в теоретико-множественную онтологию со всеми существующими для этого элемента отношениями таксономии, примерами для абсолютных понятий, доменами и областью определения - для относительных понятий. Первым входным параметром является знак элемента предметной области. Вторым входным параметром является онтология. В ходе выполнения программы явно указывается связь с scp-программой нахождения области определения отношения.
