Команда формирования теоретико-множественной онтологии для заданной предметной области предназначена для поиска всех теоретико-множественных связей элементов предметной области и формирования теоретико-множественной онтологии. Единственным входным аргументом команды является предметная область, для которой необходимо сформировать онтологию.
