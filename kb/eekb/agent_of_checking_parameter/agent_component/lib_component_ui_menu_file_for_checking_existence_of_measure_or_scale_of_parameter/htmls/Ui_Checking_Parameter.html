Команда проверки наличия единицы измерения или шкалы у параметра предназначена для выполнения поиска в базе знаний параметра, у которого отсуствуют необходимые конструкции, такие как: измерение по шкале и измерение с фиксированной единицей измерения. Аргументом запроса является контур, в котором производится поиск.
