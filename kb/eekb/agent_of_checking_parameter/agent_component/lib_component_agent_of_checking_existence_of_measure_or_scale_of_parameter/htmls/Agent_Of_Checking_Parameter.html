Задачей проверки наличия единицы измерения или шкалы у параметра является поиск в базе знаний параметра, у которого отсуствуют необходимые конструкции, такие как: измерение по шкале и измерение с фиксированной единицей измерения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия единицы измерения или шкалы у параметра. Аргументом запроса является контур, в котором производится поиск.
