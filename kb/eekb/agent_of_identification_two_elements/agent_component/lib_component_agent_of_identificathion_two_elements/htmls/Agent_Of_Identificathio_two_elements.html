Задачей агента отождествления двух элементов является поиск всех теоретико-множественных отношений в предметной области. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей поиску теоретико-множественной онтологии. Первым аргументом запроса является предметная область.
