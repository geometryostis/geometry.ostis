Задачей агента вывода всех утверждений, использующихся для доказательства исходного утверждения является поиск утверждений, имеющихся в базе знаний, и использующихся для доказательства исходного утверждения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей вывода всех утверждений, использующихся для доказательства исходного утверждения. Аргументом запроса является утверждение, для которого необходимо найти использующихся для его доказательства утверждения.
