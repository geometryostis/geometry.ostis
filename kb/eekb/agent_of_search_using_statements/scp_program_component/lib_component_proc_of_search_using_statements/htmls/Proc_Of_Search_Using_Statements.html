Задачей scp-программы вывода всех утверждений, использующихся для доказательства исходного утверждения является поиск утверждений, имеющихся в базе знаний, и использующихся для доказательства исходного утверждения. Первым входным параметром данной scp-программы является утверждение, для которого необходимо определить все использующихся для доказательства данного утверждения. Вторым входным параметром является множество, которое представляет собой узел ответа, в котором хранится созданная конструкция.
