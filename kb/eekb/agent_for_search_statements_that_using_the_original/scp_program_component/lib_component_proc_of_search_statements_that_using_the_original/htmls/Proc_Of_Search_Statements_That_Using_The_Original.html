Задачей scp-программы поиска всех утверждений, для доказательства которых используется исходное утверждение является поиск утверждений, имеющихся в базе знаний, и использующих для доказательства исходное утверждение. Первым входным параметром данной scp-программы является утверждение, которое используется для доказательства. Вторым входным параметром является множество, которое представляет собой узел ответа, в котором хранится созданная конструкция.
