Задачей вывода всех утверждений, для доказательства которых используется исходное утверждение является поиск утверждений, имеющихся в базе знаний, и использующие для доказательства исходное утверждение. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей вывода всех утверждений, для доказательства которых используется исходное утверждение. Аргументом запроса является утверждение, для которого необходимо найти утверждения, использующие его для доказательства.
