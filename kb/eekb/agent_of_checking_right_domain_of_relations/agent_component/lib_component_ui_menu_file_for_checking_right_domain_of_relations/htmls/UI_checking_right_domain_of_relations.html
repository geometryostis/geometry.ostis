Команда пользовательского интерфейса проверки элементов, участвующих в отношении, на соответствие доменам для указанного отношения предназначена для проверки правильности соотнесения доменов. Первым входным параметром является знак структуры, в рамках которой будут проверяться отношения на соответствие доменам. Результатом выполнения команды является знак структуры, являющейся ответом.
