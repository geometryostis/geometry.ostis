Задачей агента проверки элементов, участвующих в отношении, на соответствие доменам для указанного отношения является проверка элементов, участвующих в отношении, на соответствие доменам для указанного отношения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки элементов, участвующих в отношении . Первым входным параметром является знак структуры, в рамках которой будут проверяться отношения на соответствие доменам. В ходе выполнения агента явно указывается связь с scp-программой добавления конструкции с бинарной дугой в ответ, scp-программой формирования множества доменов для конструкции с бинарной ориентированной дугой, scp-программой формирования множества доменов для конструкции с бинарной неориентированной дугой, scp-программой формирования множества доменов для связки бинарного отношения, scp-программой проверки соотнесения элементов связок отношения с их доменами, scp-программой формирования множества доменов для связки бинарного отношения, scp-программой генерирования множества доменов , scp-программой формирования множества доменов для связки квазибинарного отношения, scp-программой формирования множества доменов для связки тернарного отношения, scp-программой проверки соотнесения элементов тернарного отношения с их доменами, scp-программой формирования множества доменов для связки унарного отношения, scp-программой проверки соотнесения элементов бинарного отношения с их доменами.
