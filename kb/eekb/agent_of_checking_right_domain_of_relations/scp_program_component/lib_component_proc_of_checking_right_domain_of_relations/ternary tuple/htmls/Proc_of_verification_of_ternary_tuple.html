Задачей scp-программы проверки соотнесения элементов тернарного отношения с их доменами является проверка соотнесения элементов тернарного отношения с их доменами. Первым входным параметром является знак первого проверяемого элемента. Вторым входным параметром является знак множества доменов, в рамках которого будет проверяться первый проверяемый элемент. Третьим входным параметром является знак второго проверяемого элемента. Четвертым входным параметром является знак множества доменов, в рамках которого будет проверяться второй проверяемый элемент. Пятым входным параметром является знак пятого проверяемого элемента. Шестым входным параметром является знак множества доменов, в рамках которого будет проверяться шестой проверяемый элемент.
