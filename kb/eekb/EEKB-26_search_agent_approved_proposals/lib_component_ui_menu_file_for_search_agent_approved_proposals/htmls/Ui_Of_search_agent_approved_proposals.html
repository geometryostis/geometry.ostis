Команда пользовательского интерфейса для поиска утвержденных предложений предназначена для поиска утвержденных предложений. Единственным аргументом команды является пользователь., для которого необходимо найти все утвержденные предложения. Результатом выполнения команды является структура, содержащая все утвержденные предложения.
