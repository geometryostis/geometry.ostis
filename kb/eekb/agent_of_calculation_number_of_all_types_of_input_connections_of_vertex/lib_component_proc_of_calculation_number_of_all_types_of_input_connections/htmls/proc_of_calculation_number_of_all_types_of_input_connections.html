Задачей scp-программы подсчета среднего числа типов входящих связей вершины онтологии является подсчет среднего числа типов входящих связей для заданной вершины онтологии. Первым входным параметром является множество вершин, для которых будет высчитываться среднее число типов входящих связей. Вторым входным параметром является множество sc-ссылок,при сложении которых получим общее число типов входящих связей онтологии, которое после поделим на мощность множества вершин и получим среднее число типов входящих связей вершины онтологии. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества и с scp-программой подсчета среднего значения.
