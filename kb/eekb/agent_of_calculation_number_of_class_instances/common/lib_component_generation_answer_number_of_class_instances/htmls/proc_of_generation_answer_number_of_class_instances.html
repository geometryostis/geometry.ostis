Задачей scp-программы фомироваания мощности элемента является формирование конструкции отражающей мощность, отражающей мощность данного элемента. Первым входным параметром является sc-элемент, являющийся классом, для которого должна быть сгенерирована конструкция. Вторым входным параметром является sc-ссылка, в которой должно содержаться число элементов класса. Третьим входным параметрос является множетсво, в которое занесется конструкция. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: отношениями десятичное число', измерение* и идентификация*.
