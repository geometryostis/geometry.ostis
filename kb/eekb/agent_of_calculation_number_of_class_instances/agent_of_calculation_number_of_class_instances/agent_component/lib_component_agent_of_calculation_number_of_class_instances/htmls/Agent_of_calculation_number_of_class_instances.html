Задачей агента подсчета количества элементов класса является поиск и подсчет всех элементов данного класса в системе. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества элементов класса. Результатом выполнения агента является вывод на экран мощности данного класса.
