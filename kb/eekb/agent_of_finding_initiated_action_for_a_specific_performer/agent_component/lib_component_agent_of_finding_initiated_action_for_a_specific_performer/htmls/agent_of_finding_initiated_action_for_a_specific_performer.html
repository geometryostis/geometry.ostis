Задачей агента поиска инициированных действий по исполнителю является поиск активных задач, которые были закреплены за конкретным разработчиком* с утверждения* менеджера*. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу активных задач закрепленных за конкретным исполнителем. Единственным аргументом запроса является исполнитель*, один из разработчиков* базы знаний, активные задачи которому требуется найти.
