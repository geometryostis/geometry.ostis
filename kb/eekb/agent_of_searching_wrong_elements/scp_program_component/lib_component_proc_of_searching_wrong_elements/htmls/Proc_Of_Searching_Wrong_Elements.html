Задачей scp-программы поиска sc-элементов без главного идентификатора является поиск в базе знаний элементов, у которых отсуствуют необходимые конструкции, в данном случае sc-элементов, имеющих системный идентификатор, но не имеющих ни одного основного. Входным параметром данной scp-программы является множество, которое представляет собой узел ответа, в который помещаются найденные элементы.
