Задачей агента подсчета процента временных сущностей является поиск и подсчет всех временных сущностей в системе. На вход данному агенту подается множество временных сущностей. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу нахождения процента временных сущностей. Результатом выполнения агента является вывод на экран мощности множества временных сущностей.
